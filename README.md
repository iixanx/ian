## System Required

- Node Version >= 20.11.0

## Installation

```bash
$ pnpm i
```

## Running

### Prod
```bash
$ pnpm run start
```

### Dev

```bash
$ pnpm run dev
```

### Test

```bash
$ pnpm run test # Unit Test

$ pnpm run test:e2e # E2E Test

$ pnpm run test:cov # Coverage
```
